=====

MiniG is a webmailer written in Java.

Installation
------------

**Prerequisite**

- Oracle Java 8
- Apache Maven 3.1.x or newer
- IMAP Server

**Build and package**

- run `mvn package`
- You will find a fat jar (Spring Boot application) under `target`
- run `java -jar minig.jar`
